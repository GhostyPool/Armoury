# Armoury
A helper plugin for Mortal Kombat 1 that enables new skins in the game.

This mod isn't meant to be used standalone, but rather as a dependency of other mods that register new skins.

## Installation

Drag and drop extracted files into the **Mortal Kombat 1\MK12\Binaries\Win64** folder.

You can find your game's directory in Steam by right-clicking the game->Manage->Browse local files.

### Usage

> [!IMPORTANT]
> If you are just installing this plugin because a mod requires it, the following does not concern you.

If you are looking to integrate your own skins, please head to the wiki.
